# fineRecipeFinder 🧑‍🍳

A premium, high-fidelity recipe discovery web application built as an ALX Front-End Web Development Capstone project. **fineRecipeFinder** allows users to explore thousands of global recipes, providing a seamless, restaurant-quality UI with detailed ingredient lists and step-by-step cooking instructions.

## ✨ Features

- **Premium User Interface:** A modern, dark-themed hero section with a warm, appetizing color palette built using Tailwind CSS v4.
- **Dynamic Search Engine:** Real-time search functionality that queries a vast database of meals by name or ingredient.
- **Responsive Recipe Grid:** A mobile-first, responsive masonry-style grid featuring hover effects and category tags.
- **Comprehensive Recipe Details:** Detailed recipe pages featuring high-quality imagery, categorized tags, extracted ingredient lists with exact measurements, and formatted instructions.
- **Video Integration:** Direct links to YouTube cooking tutorials for supported recipes.
- **Robust Error Handling:** Clean UI states for loading (spinners), empty results, and API errors.

## Tech Stack

- **Framework:** [React 18](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Routing:** [React Router DOM v6](https://reactrouter.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **API:** [TheMealDB](https://www.themealdb.com/api.php) (RESTful API)
